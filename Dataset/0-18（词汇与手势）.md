# 汉语沟通发展量表（普通话版）：词汇及手势

> 这个词汇表是用于不同年龄的孩子，我们请您来填这个词汇表的唯一目的就是想了解孩子们一般什么时候才开始讲话，什么时候会讲什么话。因此，您的孩子现在完全有可能只知道这表上很少的词，甚至根本不知道，这也是正常的，绝对不要为孩子究竟知道其中多少个词而担心。

---

## 第一部分：早期对语言的反应

### 甲、初期对语言的反应标

**选项**: 没有, 有

| 序号 | 题目 |
|------|------|
| 1 | 叫他的名字时会有反应（例如：转向及看声音来源的方向）。 |
| 2 | 别人说“别××/不许××”时会有反应（例如：暂停做某件事）。 |
| 3 | 当听到“妈妈/爸爸在哪里？”会向周围找。 |

### 丙、开始说话的方式

*您的孩子是否做过下面的言语行为？如果还没有或不是这样说，请填“从不”；如果做过一两次但不是天天做，请填“有时”；如果他是经常或每天都会做，请填“经常”。*

**选项**: 从不, 有时, 经常

| 序号 | 题目 |
|------|------|
| 1 | 有些孩子看到想要的东西就会用手指并发出些“嗯”的声音。您的孩子有没有这样做？ |
| 2 | 看到想要的东西就会直接说“要”/“拿”。 |
| 3 | 有些孩子喜欢“像鹦鹉般地复述”或模仿一些他们刚刚听过的声音（包括他们刚刚学到的新字，以及/部分句子，例如：当妈妈说：“我上班啦！”之后，孩子重复“班啦！”）。你的孩子有多少模仿别人说话的时候呢？ |
| 4 | 某些孩子一看到东西，就喜欢叫出它的名字。您的孩子会这样做吗？ |

### 丁、词汇量表

*当您说出该词时，孩子虽然还不能说出来，但能正确地反应，请填“听懂”。若孩子又能听懂又能够自己说出该字，请填“能说”。假如您的孩子对该字的发音不准（例如：火车变为“火些”）或他的说法有所不同（例一：葡萄说：“萄”；例二：老鼠变为“耗子”），请填“能说”。当遇到“面/面条”这种中间划有“/”的词汇时，对会说“/”两侧中的一个词即为“会说”。*

**选项**: 不懂, 听懂, 能说

## 第二部分 动作及手势

### A. 初期沟通手势

*当婴儿刚开始学习沟通的时候，他们经常用手势去表达他们的思想和愿望。选出以下哪些项目可以是您孩子平时所做的动作。（“有时”指孩子用过一次以上；“经常”指需要用的时候大部分会用到）*

**选项**: 还没有, 有时会, 经常会

| 序号 | 题目 |
|------|------|
| 1 | 举高他/她的手表示想要抱抱。 |
| 2 | 当有人离开时，做“再见”的手势。 |
| 3 | 抱拳表示“谢谢！” |
| 4 | 指（包括手臂及手指一起伸展）向一些有趣的对象或事件。 |
| 5 | 把他手中的玩具或对象交给你。 |
| 6 | “咿咿”嘴表示好吃。 |
| 7 | 伸展他/她的手臂，显示手中握着一些东西。 |
| 8 | 摇头表示“不好/不要/不想”。 |
| 9 | 摊开双手表示“没有”。 |
| 10 | 飞吻。 |
| 11 | 点头表示“好/同意”。 |
